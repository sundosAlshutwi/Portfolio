# Sundos Alshutwi — Personal Portfolio

A personal portfolio website showcasing my background in Information Technology, 
Data Analysis, and Artificial Intelligence.

## Sections
- **About** — brief introduction and background
- **Projects & Certificates** — Industrial Energy Consumption Anomaly Detection System, 
  Data Analyst Nanodegree (Udacity), and AI Course (Samsung Innovation Campus)
- **Contact** — email, phone, and LinkedIn

## Built with
- HTML5, CSS3, JavaScript (no frameworks)
- Fonts: Sora & Work Sans (Google Fonts)

## View the site
Open `portfolio-modern_7.html` in any browser, or enable GitHub Pages 
(Settings → Pages → deploy from main branch) to view it live.
